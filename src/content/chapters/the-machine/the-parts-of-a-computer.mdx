A computer is built from a small number of hardware parts.

### CPU (Central Processing Unit)

Executes instructions. Performs arithmetic, logic, and control flow. Modern CPUs have multiple *cores* — independent processors that run in parallel.

### GPU (Graphics Processing Unit)

A specialized processor for parallel work. Used for graphics, AI, and video.

### RAM (Random Access Memory)

Fast, volatile working memory. Holds the data the CPU is actively using. Loses its contents when power is cut.

### Storage (SSD / HDD)

Non-volatile memory. Keeps data when the power is off. Slower than RAM.

### Motherboard

Connects the parts. Carries data between them over buses.

### I/O (Input / Output)

Keyboard, mouse, display, network card, USB. The interface between the machine and the outside world.
